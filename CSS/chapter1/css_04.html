HTML의 개요

HTML은 HyperText Markup Language의 약자로, 웹의 표준을 주관하는 W3C에서 발표한 프로그래밍 언어이며,
웹 페이지를 제작하기 위한 기본 언어이다.

HTML로 제작된 페이지는 웹 브라우저가 해석하여 이용자들에게 보여주게 된다.
현재는 XML 토대 위에 HTML을 재구성하여 확장성과 이동성을 갖추도록 한 XHTML이 개발되어 함께 사용되고 있다.

HTML 문서의 구조

  • HTML 문서는 태그(TAG)라 불리는 코드들로 구성된다.
  • HTML 문서는 메모장과 같은 일반적인 에디터나 드림위버 같은 전문적인 웹에디터를 이용해 작성한다.
  • HTML을 이용하여 제작한 웹 페이지 문서의 확장명은 *.htm 또는 *.html이다.

HTML 태그의 종류

  1. 문단구성 관련 태그 : br, p, div
  2. 목록 관련 태그 : ol, ul, li, dl, dt, dd
  3. 문자 관련 태그 : em, strong, h1~h6
  4. 표제작 관련 태그 : table, tr, td, th, caption
태그 관련 속성들
width, height, border, rowspan, colspan
